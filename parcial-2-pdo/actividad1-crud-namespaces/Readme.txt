# CRUD PHP con Namespaces y Autoload

## Autor
Brant Santana

## Descripción
Este proyecto consiste en un sistema CRUD desarrollado en PHP utilizando programación orientada a objetos.

Se implementaron namespaces y autoload para organizar el código y cargar automáticamente las clases.

## Funcionalidades
- Crear productos
- Listar productos
- Editar productos
- Eliminar productos

## Tecnologías utilizadas
- PHP
- MySQL
- PDO

## Estructura del proyecto
- models: clases del sistema
- controllers: lógica CRUD
- config: conexión a la base de datos
- vendor: autoload
