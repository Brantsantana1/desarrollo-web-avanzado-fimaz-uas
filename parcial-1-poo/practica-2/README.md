# Práctica 2 – Herencia en PHP

## ¿Qué es la herencia?
La herencia permite que una clase hija reutilice atributos y métodos de una clase padre.
En esta práctica, la clase Admin extiende a Usuario.

## Diferencias entre Usuario y Admin
Usuario contiene nombre y correo.
Admin hereda esos atributos y agrega el método getRol() que retorna "Administrador".

## Evidencia de ejecución
El programa se ejecuta correctamente en el navegador mostrando el nombre, correo y rol del administrador sin errores.
